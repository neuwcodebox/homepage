# Nanobot 설치

OpenClaw는 좀 과한 거 같아서 다른 유사 프로젝트들을 찾아보다가 [Nanobot](https://github.com/HKUDS/nanobot)이 괜찮은 거 같아서 설정해뒀다. 그 전에는 [ZeroClaw](https://github.com/zeroclaw-labs/zeroclaw)가 괜찮아 보여서 테스트 했었는데 무한루프 돌고 제대로 동작하질 않아서 포기했다. Nanobot은 일단 기본 동작은 잘 해서 좀 더 써보려고 한다.

{/* truncate */}

OpenClaw가 핫하긴 한데 정작 딱히 쓸 곳이 없는 거 같아서 냅두고 있다가 저번에 할인 받은 카톡 ChatGPT Pro 쿠폰을 적용한 김에 Codex OAuth 로그인으로 써보려고 Nanobot을 설치해봤다. 설치도 간편하고 스무스하게 동작한다. 다만 json 파일을 직접 수정해서 설정해야 하는 건 좀 번거롭긴 했다. 릴리즈 vs 최신 코드 차이 이슈로 README랑 안 맞는 부분이 있어서 조금 삽질도 했다.

일단 디스코드랑 텔레그램에 연결해뒀다. 디스코드만 쓸 거 같긴 한데 텔그가 최근 스트리밍 응답을 지원하게 됐다고 들어서 궁금해서 해봤음.

프로그램 자체와는 별개로 오픈소스 프로젝트로서도 꽤 괜찮은 거 같다. 코드 자체가 읽기 쉽게 구성되어 있어서 나처럼 AI Agent 저수준으로 구현해야 하는 사람들은 공부 될 듯.

흠 근데 여전히 뭐에 써야 좋을지 잘 모르겠다. 많은 것들을 할 수 있다는데 내가 그걸 평소에도 했으면 모르겠는데 굳이 일을 벌리면 귀찮기만 한 게... 돈이나 대신 벌어줬으면.
