# 스텔라 코드

최근에 출시된 비주얼 노벨 게임이다. 이과 감성이 매우 충만한 SF 장르. 개인적으로 컴퓨터 공학과 무선 통신 분야에 지식이 있어서 관련 내용이 나오는 게 참 재밌었다. 그리고 천문학, 물리학과 관련된 내용도 흥미롭다. 어느정도 과학적인 내용을 쌓고 픽션을 가미하는 게 SF의 묘미인 거 같다. 참고로 [스팀](https://store.steampowered.com/app/3411510/Stellar_Code/)에서 구매했다. 공돌이 감성을 좋아한다면 추천함.
