# Claude Code

AI 코딩 도구를 몇개 써봤지만 요즘 핫한 Claude Code는 아직 써보질 않아서 이참에 써봤다.
구독을 하진 않았고 일단 사용량 과금으로 해봄.

<!-- truncate -->

생각보다 요금이 꽤 쎄다. `CLAUDE.md` 파일 작성이랑 100줄 안되는 파일 하나 수정하는 간단한 작업 맡겼는데 한화로 680원 정도 나왔다.
문제는 그 간단한 작업의 결과물이 그리 좋지 못해서 몇번 대화가 오갔고 그러면서 요금이 더 나왔고 좀 아까운 상태다.

프로젝트 경로에 `CLAUDE.md`라는 파일을 만들어서 AI에게 전반적인 가이드를 하고 메모장 느낌으로도 쓰고 그러는 거 같다.
`AGENTS.md`라는 파일로 표준화 하려는 움직임이 있고 참여한 코딩 에이전트 서비스들도 꽤 있지만 Claude는 아직이다.
`AGENTS.md` 파일을 만들고 심볼릭 링크로 `CLAUDE.md` 만들면 되기는 하는데 이것도 완벽한 방법은 아니라 함.

아무튼 아직 뭐라 평가하긴 이르지만 첫인상은 별로다. Claude는 코딩을 잘하는 걸로 알고 있고 실제로 웹 페이지에서 써봤을 때에도 그렇게 느꼈는데... 뭐 몇번 더 써보면 알겠지.

그리고 CLI에서 쓰는 게 장점도 있겠지만 GUI로 써보고 싶어서 찾아보니 [opcode](https://github.com/getAsterisk/opcode)라는 게 있더라. 근데 Windows는 직접 컴파일해서 써야 함... ㅠ

아무래도 무료 요금제가 없고 비용이 꽤 높아서 평소에 쓰지는 않을 거 같다. 대신 Gemini CLI, GitHub Copilot 같은 거 쓰다가 잘 안 되면 쓸 듯.

요즘 AI 도구가 우후죽순 나오는데 승자는 누가 될 것인지 궁금해진다.
