# 2026-05-21 - label

![label 이미지](https://cdn.neuwappbox.com/docs/today-en-word/2026-05-21-label-34954130237b.webp)

* **발음**: /ˈleɪbəl/
* **품사**: 명사, 동사
* **의미**
  * 명사: 라벨, 표지, 꼬리표; 분류명, 딱지
  * 동사: 라벨을 붙이다; 어떤 사람이나 사물을 특정 방식으로 규정하다
* **형태**
  * 복수: *labels*
  * 과거형/과거분사: *labeled* 또는 *labelled*
  * 현재분사: *labeling* 또는 *labelling*
  * 미국 영어에서는 보통 *labeled*, *labeling*을 더 많이 쓰고, 영국 영어에서는 *labelled*, *labelling*도 흔합니다.
* **예문**
  * *Please read the label before taking the medicine.*
    약을 복용하기 전에 라벨을 읽어 주세요.
  * *She labeled each box with its contents.*
    그녀는 각 상자에 내용물을 적은 라벨을 붙였습니다.
  * *It is unfair to label someone as lazy after one mistake.*
    한 번의 실수만으로 누군가를 게으르다고 단정하는 것은 불공평합니다.
* **자주 쓰는 표현**
  * *warning label*: 경고 라벨
  * *price label*: 가격표
  * *record label*: 음반사
  * *private label*: 자체 상표, 유통업체 브랜드
  * *label someone as something*: 누군가를 무엇이라고 규정하다
* **유의어/관련어**
  * *tag*: 꼬리표, 태그
  * *sticker*: 스티커
  * *caption*: 설명문, 자막
  * *brand*: 상표, 브랜드
  * *category*: 범주, 분류
* **활용 팁**
  * 물건에 붙은 실제 표시는 *label*, 줄이나 끈으로 매단 꼬리표는 *tag*가 더 자연스러운 경우가 많습니다.
  * 사람에게 쓰일 때는 조심해야 합니다. *label someone*은 단순히 이름표를 붙인다는 뜻을 넘어 "성급하게 판단하다"라는 부정적인 뉘앙스를 가질 수 있습니다.
  * 식품이나 의약품 문맥에서는 *read the label*이 매우 자연스러운 표현입니다.
* **어원**
  * *label*은 중세 프랑스어 *label*에서 들어온 말로, 원래는 옷이나 문서에 붙는 작은 조각, 끈, 표식을 가리키는 말이었습니다.
* **트리비아**
  * 음악 업계의 *record label*은 원래 음반 가운데 붙어 있던 종이 라벨에서 나온 표현입니다. 시간이 지나면서 음반을 제작하고 유통하는 회사 자체를 뜻하게 되었습니다.
