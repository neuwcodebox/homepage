# 2026-07-11 - react

![react 이미지](https://cdn.neuwappbox.com/docs/today-en-word/2026-07-11-react-235638c340e4.webp)

* **발음**: /riˈækt/ 또는 "리액트"
* **품사**: 동사
* **의미**
  * 어떤 일, 말, 자극에 반응하다
  * 화학 물질 등이 반응하다
  * 어떤 상황에 좋게 또는 나쁘게 반응하다
* **형태**
  * 3인칭 단수: *reacts*
  * 과거형/과거분사: *reacted*
  * 현재분사: *reacting*
  * 명사형: *reaction* 반응
  * 형용사형: *reactive* 반응하는, 반응성이 있는
* **예문**
  * *She reacted calmly to the unexpected question.*  
    그녀는 예상치 못한 질문에 침착하게 반응했다.
  * *Plants react to changes in light and temperature.*  
    식물은 빛과 온도의 변화에 반응한다.
  * *The two chemicals reacted and produced heat.*  
    두 화학 물질이 반응하여 열을 냈다.
* **자주 쓰는 표현**
  * **react to + 명사**: 무엇에 반응하다
    * *How did he react to the news?* 뉴스에 그는 어떻게 반응했나요?
  * **react with + 명사**: 무엇과 화학적으로 반응하다
    * *Iron reacts with oxygen.* 철은 산소와 반응한다.
  * **react badly/well**: 나쁘게/좋게 반응하다
  * **overreact**: 지나치게 반응하다
* **유의어/관련어**
  * **respond**: 반응하다, 응답하다. 말이나 요청에 답하는 뜻으로도 매우 자주 씁니다.
  * **reply**: 말이나 글로 대답하다. 보통 사람의 응답에 씁니다.
  * **respond to**: *react to*보다 조금 더 중립적이고 의도적인 반응을 나타낼 수 있습니다.
  * **reaction**, **responsive**, **interaction**
* **활용 팁**
  * 일상에서는 대개 **react to**를 씁니다. 뒤에는 사건, 말, 표정, 소식처럼 반응의 대상이 옵니다.
  * *react*는 즉각적이거나 감정적인 반응을 떠올리게 하는 경우가 많습니다. 반면 *respond*는 보다 차분하고 의식적인 대응에도 잘 어울립니다.
  * 형용사 앞에서는 *react positively*, *react angrily*처럼 부사와 함께 쓸 수 있습니다.
* **어원**
  * *react*는 라틴어 *re-* "다시, 뒤로"와 *agere* "행동하다, 하다"에서 온 말입니다. 처음에는 어떤 작용에 맞서 다시 작용한다는 의미였고, 오늘날에는 사람의 감정적 반응부터 화학 반응까지 폭넓게 쓰입니다.
* **트리비아**
  * 과학에서 *reaction*은 단순한 감정 표현이 아니라 물질이 서로 만나 성질이 바뀌는 과정을 뜻합니다. 그래서 *chemical reaction*은 "화학 반응"입니다.
  * **reaction time**은 자극을 인지한 뒤 행동으로 옮기기까지 걸리는 "반응 시간"을 말합니다.
