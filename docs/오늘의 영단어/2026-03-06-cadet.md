# 2026-03-06 - cadet

* **발음**: /kəˈdet/
* **품사**: 명사
* **의미**:
  * 사관학교 생도, 후보생
  * (확장) 특정 조직에서 훈련 중인 초급 인원
* **형태**:
  * 단수: cadet
  * 복수: cadets
* **예문**:
  * *The cadet woke up at dawn for physical training.*  
    그 생도는 체력 훈련을 위해 새벽에 일어났다.
  * *She entered the academy as a cadet at nineteen.*  
    그녀는 열아홉 살에 생도로 사관학교에 입학했다.
  * *The senior officer advised each cadet to build discipline first.*  
    선임 장교는 각 생도에게 먼저 규율을 세우라고 조언했다.
* **유의어/관련어**: *trainee*, *recruit*, *midshipman* (해군 맥락)
* **활용**:
  * 군사 맥락에서는 사관학교 학생을 뜻할 때 가장 자연스럽습니다.
  * 비군사 맥락에서도 경찰학교, 항공훈련기관 등에서 훈련생 의미로 확장되어 쓰입니다.
* **어원**:
  * 프랑스어 *cadet*에서 왔으며, 원래는 집안의 "막내"를 뜻하던 말이었습니다.
  * 이후 "젊은 후보자, 수습 인원" 의미로 발전해 현대 영어의 의미가 되었습니다.
* **트리비아**:
  * 영어 표현 *cadet branch*는 가문의 본류가 아닌 방계 혈통을 뜻합니다.
  * 단어의 핵심 뉘앙스는 "아직 정식 역할 이전의 훈련 단계"입니다.
