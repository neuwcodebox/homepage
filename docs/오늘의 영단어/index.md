# 2025-11-27 - index

* **발음**: /ˈɪn.dɛks/
* **품사**: 명사, 동사
* **의미**:
  * (명사) 색인; 지표·지수; (금융) 여러 종목의 평균을 나타내는 지수
  * (동사) 색인을 만들다; (데이터·가격 등을) 지수화하다
* **형태**
  * 복수: indexes 또는 indices (둘 다 흔히 사용)
  * 동사 변화: indexes – indexed – indexing
* **예문**
  * *Please check the index at the back of the book to find the chapter on climate change.*  
    책 뒤쪽의 색인을 확인해 기후 변화 장을 찾아보세요.
  * *The CPI is a key index that tracks consumer price changes over time.*  
    소비자물가지수(CPI)는 시간이 지나며 소비자 물가 변화를 추적하는 핵심 지표입니다.
  * *They indexed all the research papers so the database could be searched faster.*  
    연구 논문을 모두 색인화하여 데이터베이스를 더 빠르게 검색할 수 있게 했다.
* **유의어/관련어**: catalogue, register, indicator, gauge, benchmark
* **활용 팁**
  * 학술·출판에서 책의 색인을 가리킬 때는 주로 index(복수 indexes), 수학·경제에서 통계 지표를 말할 때는 index의 복수형 indices도 자주 사용됩니다.
  * 데이터베이스에서 	o index는 검색 성능을 높이기 위해 키를 생성한다는 의미로 쓰입니다.
* **어원**: 라틴어 *index* "지시자, 표시"에서 유래. *in-* (안쪽) + *dicere* (말하다, 지시하다)에서 파생.
* **트리비아**: 로마 시대에는 법정에서 발언 순서를 알려주는 손짓을 가리키는 말로 쓰였고, 현대 금융에서는 S&P 500 같은 주가지수를 대표하는 용어가 되었습니다.
