# 2026-01-13 - array

* **발음**: /uh-RAY/
* **품사**: 명사, 동사
* **의미**
  * (명사) 배열, 정렬; 다양한 것들의 집합
  * (동사) (보기 좋게) 정렬하다, 배치하다
* **형태**
  * 명사 복수: arrays
  * 동사 활용: array - arrays - arrayed - arraying
* **예문**
  * *The shop offers an array of seasonal fruits.*  
    그 가게는 제철 과일의 다양한 종류를 제공합니다.
  * *Please array the documents by date.*  
    문서를 날짜순으로 정렬해 주세요.
  * *An array of sensors monitors the engine.*  
    센서들의 집합이 엔진을 모니터링합니다.
* **유의어/관련어**: assortment, range, collection, lineup
* **활용**
  * **an array of + 복수 명사** 형태로 "다양한 ~"라는 뜻을 가장 자주 만듭니다.
  * 동사로는 격식체에서 "정렬하다, 늘어놓다" 의미로 쓰입니다.
* **어원/트리비아**
  * 프랑스어 *arrer* (정렬하다)에서 유래했습니다.
  * 컴퓨터 과학에서는 "배열"로 번역되며, 같은 종류의 데이터를 순서대로 저장하는 구조를 뜻합니다.
